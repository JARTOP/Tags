# Tag & Screen System

## Описание
Система управления тегами (Tag Service) и экранами (Screen Service) для отображения графиков в реальном времени.

### Характеристики
- **2000 тегов** с обновлением каждые 50 мс
- **30 тегов** обновляются каждые 17 мс
- **3-12 экранов**, каждый с 300-500 тегами
- **2-5 экранов-графиков** с формулой y ∈ [-2,0]
- Автообновление экранов каждые 250 мс
- **Queue Size** (1-500) для просмотра последних N значений
- **Поиск по дате** на графике
- Роли: **Admin** (редактирование) и **Operator** (только просмотр)
- База данных в .txt файле
