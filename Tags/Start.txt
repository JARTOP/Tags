Как собрать и запустить:
Создайте новое консольное приложение:

dotnet new console -n TagScreenSystem
cd TagScreenSystem
Создайте указанную структуру папок и добавьте все файлы

Запустите проект:
dotnet run
Особенности реализации:
Разделение ответственности - каждая система в своем файле

Оптимизация БД - буферизация с пакетной записью

Интерфейс в стиле ASCII - красивое отображение в консоли

Два режима пользователя - Админ может редактировать теги

Автообновление - теги (50 мс), экраны (250 мс)

Графики с ползунком - навигация стрелками и поиск по дате
